# ❤️ Bayesian Heart Disease Prediction

## 📌 Overview
This project predicts the likelihood of heart disease using a probabilistic approach based on **Bayes’ Theorem**. It uses the **Naive Bayes algorithm** to analyze patient medical data and provides both predictions and probability outputs.

---

## 🎯 Objective
- Predict whether a patient has heart disease  
- Calculate the probability of disease  
- Perform data analysis and visualization  

---

## 🧠 Concept
The model is based on Bayes’ Theorem:

P(D|X) = (P(X|D) * P(D)) / P(X)

Where:
- D = Disease  
- X = Patient features  

Naive Bayes applies this by assuming independence between features.

---

## 📊 Dataset
- Heart Disease UCI Dataset  
- Features include:
  - Age  
  - Sex  
  - Chest pain type  
  - Blood pressure  
  - Cholesterol  
  - ECG results  
  - Exercise-induced angina
